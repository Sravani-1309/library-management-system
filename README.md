# 📚 Library Management System

A full-stack web application built using **Python Flask and MySQL** to manage books in a library.

The system allows administrators to add books, issue books to students, return books, and automatically track overdue books with calculated fines.

---

# 🚀 Features

* Add and delete books
* Search books by title
* Issue books to students
* Automatic **due date generation (10 days)**
* Return book functionality
* Overdue detection
* Late days and fine calculation
* Dashboard showing total books and issued books
